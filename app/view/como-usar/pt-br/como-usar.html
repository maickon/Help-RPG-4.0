Como usar o site Help Rpg Fichas

Após feito o seu cadastro no site, você vai encontrar opções para diversos tipos de cadastro. Estes cadastros podem ser classificados conforme o sistema de RPG disponível no site. Assim você poderá fazer um cadastro de alguma arma ou armadura específica para utilizar em sua campanha ou simplesmente pesquisar pelo mesmo tipo de item feito por outros jogadores.

Na página de cadastro você vai perceber que todos os possíveis campos de entrada estão habilitados para os diversos tipos de sistema. Isso significa que você não é obrigado a preencher todos os campos do cadastro. Normalmente por padrão, somente o termo nome é obrigatório no ato do cadastro.

Além de cadastros feitos através por um longo formulário conforme o sistema de RPG escolhido. Você também tem a sua disposição uma página de cadastro mais rápida, pois somente alguns campos devem ser preenchido. A ficha por completa é informada num único campo de texto com opções para formatação. Dessa forma o formulário não fica tão extenso cheio de campos e você pode concentrar o cadastro de sua ficha em um único campo. É como se estivesse criando uma ficha no world.

Caso você deseje, poderá informar o link de uma ficha externa que você tenha registrado em algum outro lugar e cadastra-la no help RPG.

Você perceberá também que alguns cadastros como armas, armaduras e etc vão oferecer um campo de imagem para que você detalhe melhor o item na qual esteja cadastrando. Este campo não é obrigatório, caso deixe vazio o seu item cadastrado não terá imagem. Outra observação é que quando você for editar um item, você deverá fornecer novamente uma imagem, pois caso não a preencha, o seu registro ficará sem imagem.

Você vai perceber também que dentro do painel administrativo, alguns registros só estarão disponíveis para você visualizar ou baixar. Isso significa que aquele item cadastrado não é seu e somente o usuário que o cadastrou pode edita-lo. Entretanto você poderá contacta-lo através de links de referência caso o usuário tenho deixado disponívelem seu perfil para troca de informações com outros jogadores.

Todos os registros no Help RPG tem uma área reservada para comentários. Dessa forma você pode deixar a sua opinião sobre uma ficha, espada, armadura ou etc que você visualisou no site.

Algumas perguntas frequentes.

Abaixo uma pequena lista de perguntas frequentes que os usuários podem ter e possíveis sugestãos para o site.

Existe limite de itens, fichas e etc? posso cadastrar o quanto eu quiser?

Sim! No Help Rpg Fichas não existe limite para o npumero de cadastro de armas, itens, fichas e etc que você venha cadastrar. Na verdade desque não estore o limite que o plano de hospedagem oferece, esta tudo OK! rsrs

Esqueci minha senha, como recupero?

Caso você tenha esquecido a sua senha, envie um email para (helprpg.br@gmail.com) com o assunto perdi minha senha que retornaremos o mais rápido possível com uma nova senha para você.

Tem como implementar suporte a mais sistemas de RPG?

Sim! tem como. Entretanto eu não conheço todos os sistemas de RPG, portanto na medida do possível eu tentarei adicionar novos sistemas ao site para que os usuários possam cadastrar novos registros para o seu sistema de RPG preferido. O interessante é que me envie dicas de como as coisas poderiam ser feitas para um determinado sistema. Ex: Quais campos são importantes para se cadastrar e etc.

Tem como implementar novas funcionalidades?

Sim! tem como. Mas isso depende da complexidade da coisa. Infelizmente não tenho tempo para me dedicar ao site 100%. Além disso tem coisas que talvez sejam inviáveis neste momento para mim, certas funcionalidades podem ser simples de serem implementadas e outras não. Dessa forma tudo dependerá da complexidade da nova funcionalidade sugerida.
